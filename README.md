# Ubuntu16.04_for_RaspberryPi

## 概要
このリポジトリはRaspberryPi 3B+に対応したUbuntu16.04のイメージデータ置き場です。

## 動作確認済みモデル
* RaspberryPi 3B+
* RaspberryPi 3B

公式イメージからの変更点は以下のサイトにまとめてあります。

https://www.asrobot.me/entry/2018/07/11/001603
